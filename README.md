# Nyx AI Platform 

Nyx is an elegant, high-performance, single-screen intelligent logic sandbox, telemetry logging suite, and enterprise agent simulation dashboard. Designed to visualize, test, and host multi-agent cognitive architecture, it provides deep compliance telemetry, flat annual dynamic tariff conversions, and real-time interface controls in a unified light-theme modern viewport.

---

##  How to Run the App Locally

Since Nyx is a client-side Single-Page Application (SPA) powered by **React 19**, **Vite 6**, and **Tailwind CSS**, you do **not** need a separate terminal backend running to view or use the simulator. Everything runs securely and instantly directly inside the browser.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Steps to Run
1. **Clone or Download** your files to a local folder.
2. **Open your Terminal** inside that folder.
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Launch the development server**:
   ```bash
   npm run dev
   ```
5. **Open the App**: Click the local address shown in your terminal (usually `http://localhost:3000` or `http://localhost:5173`).

## 🛠️ Feature Overview & Button Controls

The interface is structured in cohesive design modules. Here is exactly what they do:

### 1. Header Navigation & Brand Identity
- **Logo Block**: Visualized by a beautiful, responsive dark saffron **Crescent Moon** and a shimmering micro-pulse accent representing **Nyx** (the Greek goddess of Night & Cosmic Order). Hovering over the logo triggers an elegant spring rotation and glowing drop shadow.
- **Navigation Controls**: Frictionless jump links routing you directly to features, pricing structures, system telemetry, and resource vaults.

### 2. Intelligent Logic Canvas
An interactive sandbox representing sequential prompt-scrubbing routes and agent pipelines.
- **"Simulate Flow" Button**: Triggers a micro-transaction flow signal. Clicking it sends a glowing pulse from the **File Trigger**, through the **PII Sanitizer** and **LLM Enclave**, finishing at the **Instant Output** terminus. The dynamic **Signal Log** updates live with milliseconds telemetry logs.
- **"Add Custom Node" Button**: Appends new test orchestration nodes into your pipeline state, illustrating system modularity in real-time.

### 3. Telemetry Performance Dashboard
Real-time diagnostic analytics tracking system throughput under high enterprise workloads.
- **System Load Gauge**: Circular SVG dial showing active CPU loads. Toggles auto-scaling nodes in a continuous animation cycle.
- **SLA Response Indicators**: Staggered matrix visualizer tracking continuous server availability benchmarks across **US East** and **EU West** zones.
- **Growth Vector Line Chart**: An elegant, custom SVG curve visualizing system efficiency. Hovering over the coordinates displays precise analytical tooltips representing regional nodes.

### 4. Dynamic Financial Matrix (Pricing)
Dynamic pricing plans adjusted seamlessly via global currency values and discount structures.
- **Billing Switcher (Monthly/Annual)**: Toggles local flat-rate price computations instantly. Choosing "Annual" triggers a secure `-20%` flat discount deduction.
- **Currency Dropdown Selector**: Converts numerical tiers on the fly between **USD**, **EUR**, **GBP**, and **JPY** without forcing a slow page reload.

### 5. Categorized Support Portal (FAQ)
An interactive help desk utilizing category filters to find swift integration tutorials.
- **Category Selectors**: Switch between **Overview**, **Protocols**, **SLA**, and **Storage** filters.
- **Accordion Panels**: Expand individual topics with clean height transitions, smooth indicator rotations, and dedicated developer resources.
